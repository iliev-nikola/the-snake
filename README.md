# The Snake

## My version of the game "The Snake"

### Link to hosted project: [here](https://iliev-nikola.github.io/the-snake/)

### Technologies:

- Languages: Vanilla JS, HTML & CSS

- Change game speed any time
- Change game type (level)
- Change theme
